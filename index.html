HOW TO INSTALL
01
DOWNLOAD nexy_plugin.lua
Get the plugin file below and save it to your Roblox Plugins folder:
%LOCALAPPDATA%\Roblox\Plugins\
02
ENABLE HTTP REQUESTS
In Roblox Studio: Game Settings → Security → Allow HTTP Requests ✓
03
ENTER YOUR API KEY
Open the Nexy panel in Studio and paste your Gemini API key in the settings field.
04
BUILD WITH AI
Describe what you want, click Insert, and Nexy drops the script right into your game tree.
